# โฟลเดอร์สำหรับรูปผลงาน

วางไฟล์รูปถ่ายงานจริงที่นี่ (`public/works/`) แล้วไปใส่พาธในไฟล์
`components/Works.tsx` ช่อง `images` ของแต่ละงาน

## ตั้งชื่อไฟล์แนะนำ

ใช้รูปแบบ `<ประเภทงาน>-<ลำดับงาน>-<ลำดับรูป>.jpg`

```
install-01-01.jpg   # งานติดตั้งงานที่ 1 รูปที่ 1
install-01-02.jpg   # งานติดตั้งงานที่ 1 รูปที่ 2
install-01-03.jpg   # งานติดตั้งงานที่ 1 รูปที่ 3
clean-01-01.jpg     # งานล้างงานที่ 1 รูปที่ 1
repair-01-01.jpg
relocate-01-01.jpg
```

## ใส่รูปลงในโค้ด

เปิด `components/Works.tsx` แล้วใส่ช่อง `images` แบบอาร์เรย์ เช่น

```ts
{
  category: "install",
  title: "ติดตั้งแอร์ผนังใหม่ 2 ตัว",
  // ...
  images: [
    "/works/install-01-01.jpg",
    "/works/install-01-02.jpg",
    "/works/install-01-03.jpg",
  ],
},
```

- รูปแรกในอาร์เรย์ = รูปหน้าปกที่แสดงในการ์ด
- ถ้ามีหลายรูป บน card จะมี badge "X รูป" มุมขวาล่าง
- ลูกค้ากดที่รูปหน้าปก → เปิด lightbox ดูรูปทั้งหมดทีละรูป

## ข้อแนะนำเรื่องไฟล์

- นามสกุล: `.jpg`, `.jpeg`, `.png`, `.webp` ใช้ได้หมด (แนะนำ `.webp` สำหรับไฟล์ขนาดเล็ก)
- ขนาด: กว้าง 1200–1600px กำลังดี (ไม่ต้องใหญ่กว่านี้)
- สัดส่วน: 4:3 หรือ 16:9 บนการ์ดจะครอปเป็น 4:3 อัตโนมัติ
- ลดขนาดไฟล์ก่อนอัปโหลด (เช่น < 300KB/รูป) จะโหลดเร็วกว่า

## การเปิดใช้ lightbox

`components/Works.tsx` รองรับอยู่แล้ว:

- คลิกที่รูปหน้าปก → เปิด lightbox เต็มจอ
- ปุ่มลูกศร ← → เลื่อนรูป หรือใช้คีย์บอร์ดก็ได้
- ปุ่ม X หรือกด Esc เพื่อปิด
- มี thumbnail strip ด้านล่างให้เลือกกระโดดดูรูปที่ต้องการ
